# Beetle Manager PWA

Next.js + TypeScript + Zustand で作成した昆虫管理アプリです。

## 機能

- 個体登録: 名前、種類、産地、温度、ステージ
- ステージ管理: 卵、幼虫、蛹、成虫
- 履歴ログ: 交換、産卵、観察、給餌、その他
- 編集と削除
- ローカルストレージ保存
- スマホ向けUI

## 今後の拡張ロードマップ

- 🌙 **ダークモード対応**: 昆虫観察に適したダークテーマの実装
- 🌳 **家系図機能**: 親個体から子個体への血統ツリー表示
- 🔔 **プッシュ通知**: エサ交換、羽化予想のリマインド機能
- 🌐 **オフライン同期**: Service Worker によるオフライン動作の強化
- 📊 **分析機能**: 成長曲線のレコード比較、羽化サイズ予測メーター
- 📱 **IoT連携**: スマート温湿度計とのライブデータ連携強化

## 開発

```bash
npm install
npm run dev
```
